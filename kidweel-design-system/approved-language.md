# Kidweel Approved Public Language

## Use

- Agents assist. Humans decide. The process records why.
- Human decision-making inside bounded delegation.
- Bounded delegation for enterprise workflows.
- Live paper-trading validation, no live capital authority.

## Avoid

- autonomous agents
- AI-powered productivity
- agentic workflows
- unlock / unleash
- signal service
- live trading
- can later enable live trade
- constraint-first (as repeated prose)
