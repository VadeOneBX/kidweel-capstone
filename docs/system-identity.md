# System identity

A **deterministic, paper-only** options decision system with a **bounded** probabilistic context layer (Claude overlay and research comparisons). It removes structurally invalid or context-degraded setups before any transport handoff—not by predicting markets, but by enforcing playbook, RR, and PMP constraints.

**Claude proposes** (interpretation, memos, research filters). **The system decides** (gates, approval, payloads). **Transport executes** (mock offline today; future paper MCP only).

**Constraint Survivability:** AI can generate actions; constraint systems determine which actions are allowed. Automation creates leverage; constraints determine whether that leverage survives scale.

The problem is not getting systems to act. It is getting them to act consistently.
